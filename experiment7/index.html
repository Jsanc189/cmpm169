Experiment 7 - Data Visualization and Networks

Imitate

For this piece, I wanted to mimic the visualization of an example seen in class.

This is the example: https://informationisbeautiful.net/visualizations/worlds-biggest-data-breaches-hacks/.

As there was no code to look at, I had to start from scratch to try and achieve at least the likeness of circles representing the data

Integrate

The first task was to provide the data for the task. I made a CSV file based on the total hours I have played for various games in my Steam Library

I then started by making circles with sizes based on the hours played per game.

Innovate

I ended up populating a dictionary with the values in the CSV file and then iterating through the dictionary frome largest to smallest hours.

I then placed each circle onto the canvas from largest to smallest, making sure that they did not overlap, and were able to touch.

I added in a "gamer flare" with color and images.

Reflection

This project may have seemed simple, but there was a lot of trouble shooting with the sizes, placement and text of each circle.

Thinking back I struggled the most with having the circles populate from largest to smallest to be able to fill the canvas without it them overlapping, all while not making the canvas look sparse.

Overall, I am happy with the result and the tone of the piece.
